# Waste-Detection-System-using-UAV-Images
Waste detection using UAV images with AI and Streamlit

## Overview
This project detects waste from drone images using AI and displays results through a web dashboard.

## Features
- Waste detection using deep learning
- Location tracking
- Map visualization
- Dashboard insights

## Tech Stack
- Python
- Streamlit
- PyTorch
- OpenCV
- Plotly
- Folium

## Run Project
pip install -r requirements.txt  
streamlit run app.py
